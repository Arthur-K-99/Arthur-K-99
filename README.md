# Athanasios Kompouras

I build hands-on labs and practical tools around network engineering, cloud infrastructure, automation, security, and scientific data. My repositories focus on reproducible environments, thorough documentation, and connecting lab concepts to real operational patterns.

## Current Focus

- Data center networking with spine-leaf fabrics, eBGP, EVPN, and VXLAN
- Network automation using Containerlab, NetBox, Ansible, and Jinja2
- Cloud architecture and infrastructure as code with Azure and Terraform
- Private PKI, ACME certificate automation, TLS, and Docker-based infrastructure
- Python applications that integrate and present complex data
